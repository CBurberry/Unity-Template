# Unity-Template
A Pre-Configured Unity Project (6.0) with common dependencies/plugins (MIT licensed) including some common source code.
For use in community projects e.g. game jams that should not be constrained by license restrictions.

Includes the following pre-configured plugins:
* [DOTween](https://assetstore.unity.com/packages/tools/animation/dotween-hotween-v2-27676)
* [NaughtyAttributes](https://assetstore.unity.com/packages/tools/utilities/naughtyattributes-129996)
